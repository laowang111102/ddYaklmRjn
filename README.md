# 旅游推荐系统 python1

## 项目概述

本项目是一款基于Python的旅游推荐系统，旨在通过机器学习算法为用户提供智能化的旅游推荐服务。系统经过二次开发，具备数据聚合分析、分页加载等新功能，并已更新至2025年3月的旅游数据。

## 技术栈

- **后端：** Django
- **开发语言：** Python
- **数据库：** MySQL
- **爬虫技术：** requests
- **推荐算法：** 机器学习
- **前端：** BootStrap + ECharts

## 功能模块

### 用户系统

- 登录
- 注册
- 个人信息管理

### 数据展示

- 旅游景点信息可视化

### 用户互动

- 景点评论
- 评分
- 反馈

### 算法推荐

- 基于机器学习智能推荐景点

### 数据分析

- 多维度统计旅游信息

### 可视化图表

- ECharts 图表展示

## 系统角色

- 用户：访问系统，获取推荐景点，参与互动等
- 管理员：负责系统维护、数据分析及用户管理等

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- requests
- ECharts
